快速熟悉 markdown

Oct 2, 2018

快速熟悉
markdown

我们可以使用 大于号 ，来表示引用

我们在文本两边，加上一对双波浪号，就得到删除特效

方框内部是文本,点击跳转百度

倘若我先写三个井号，代表我要开始列表，*、+号带领无序，1.带领有序

###

  • k1
  • k2
  • k3
  1. l1
  2. l2
  3. l3
    ###
    一对反引号可以生成一个背景加深，这很讨喜
    且表格的创建，格外方便，只用添加一点 |:—-: |这表明，我正在写表格
    |电影|导演|评分|
    |:—-:|:—-:|:—-:|
    |天注定|贾樟柯|未知|
    |江湖儿女|贾樟柯|未知|
    |三峡好人|贾樟柯|未知|
    |站台|贾樟柯|未知|

我们规定用三个反引号括住代码块
